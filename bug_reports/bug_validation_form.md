# Баг-репорт: Отсутствие валидации полей в форме оформления заказа

ID: BUG-001  
Проект: SauceDemo  
Модуль: Оформление заказа  
Дата: 21.01.2026  
Автор: Анна 
Приоритет: Высокий  
Серьезность: Критическая  

---

## Описание
Форма оформления заказа не проверяет корректность вводимых данных. Система принимает некорректные значения в полях "Имя", "Фамилия" и "Почтовый индекс".

## Шаги воспроизведения
1. Авторизоваться как standard_user / secret_sauce
2. Добавить товар в корзину → Перейти в корзину → "CHECKOUT"
3. В форме ввести:
   - First Name: 12345
   - Last Name:       (5 пробелов)
   - ZIP Code:     (3 пробела)
4. Нажать "CONTINUE"

## Фактический результат
- Форма принимает некорректные данные
- Заказ успешно оформляется
- Нет сообщений об ошибках

## Ожидаемый результат
- Форма должна проверять вводимые данные
- При некорректных данных должны быть сообщения об ошибках
- Заказ не должен оформляться с невалидными данными
