## АВТОРИЗАЦИЯ

### AUTH-001: Успешный вход стандартного пользователя
Предусловия: Открыта страница https://www.saucedemo.com/
Шаги:
1. В поле "Username" ввести "standard_user"
2. В поле "Password" ввести "secret_sauce"
3. Нажать кнопку "LOGIN"
Ожидаемый результат:
- Редирект на /inventory.html
- Отображается заголовок "Products"
- Видно 6 карточек товаров
Статус: ✅ PASS
Примечание: Основной сценарий использования

### AUTH-002: Попытка входа заблокированного пользователя
Предусловия: Открыта страница авторизации
Шаги:
1. Ввести "locked_out_user"
2. Ввести "secret_sauce"
3. Нажать "LOGIN"
Ожидаемый результат:
- Сообщение об ошибке "Epic sadface: Sorry, this user has been locked out."
- Остаемся на странице авторизации
Статус: ✅ PASS
Примечание: Проверка обработки заблокированных аккаунтов

### AUTH-003: Вход с пустым именем пользователя
Предусловия: Открыта страница авторизации
Шаги:
1. Оставить "Username" пустым
2. Ввести "secret_sauce"
3. Нажать "LOGIN"
Ожидаемый результат:
- Сообщение "Epic sadface: Username is required"
- Красный крестик рядом с полем Username
Статус: ✅ PASS
